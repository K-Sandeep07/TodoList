# To-Do List Application

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript.  
It allows users to add tasks, mark them as completed, delete tasks, and save them using localStorage so data remains even after refresh.

## Features

- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Save tasks to browser localStorage  
- Automatically loads saved tasks on refresh  
- Clean UI made with CSS and Bootstrap

## Project Structure

.
├── index.html      # Main UI  
├── index.css       # Styles for the To-Do list  
└── index.js        # All functionality & localStorage logic  

## How It Works

1. Enter a task in the input field  
2. Click "Add" to add it  
3. Check the box to mark the task completed  
4. Click the trash icon to delete  
5. Click "Save" to store tasks permanently
